单链表结构

单链表是一种链式存取的数据结构，用一组地址任意的存储单元存放线性表中的数据元素。链表中的数据是以结点来表示的，每个结点的构成：元素(数据元素的映象) + 指针(指示后继元素存储位置)，元素就是存储数据的存储单元，指针就是连接每个结点的地址数据。

单链表的节点

单链表节点的结构如下图所示，它包含了两个部分:

  • 一个数据项
  • 到结构中下一个节点的链接

    • 使用Python表示单链表的节点: 

    class Node(object):
    def __init__(self, data, next):
    self.data = data
    self.next = next
    使用单链表节点类

    节点变量可初始化为None或一个新的node对象

    node1 = None
node2 = Node("A", None)
node3 = Node("B", node2)

    以上三个节点的状态如图:

    以上的三个节点:

  • node1没有指向节点对象
  • node2和node3指向了所链接的对象
  • node2指向对象的下一个指针为None

  • probe指向空链接，表明没有搜索到目标项
  • 等于目标项的一个数据项，表明搜索成功
